# Project Revive

Project Revive is a private repository for original IGI-2-related assets created by our team, including maps, executables, sounds, and images.

## Repository Layout

- [assets/audio](assets/audio) - sound assets created by us.
- [assets/executables](assets/executables) - build outputs and executables created by us.
- [assets/images](assets/images) - image assets created by us.
- [assets/maps](assets/maps) - IGI-2 map files and related level assets created by us.
- [docs](docs) - repository documentation and supporting notes.

## Working With The Repository

- Keep all original source materials organized by asset type.
- Do not add third-party content unless its license is compatible and documented.
- Do not publish modified versions of CC BY-ND assets without explicit permission.
- Prefer descriptive filenames and keep related source notes next to the asset when useful.
