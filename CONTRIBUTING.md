# Contributing to Project Revive

Thanks for helping maintain this repository.

## Scope

This repository is for original assets created by our team for IGI-2-related work.
Contributions should stay within that scope.

## Rules

- Only contribute content you created or have explicit permission to redistribute.
- Do not submit modified derivatives of existing CC BY-ND assets unless you have written permission from the author.
- Do not add copyrighted third-party material without a clear license and attribution record.
- Keep changes organized by asset type under the existing `assets/` structure.
- Include a short note explaining what was added, changed, or replaced.

## Suggested Workflow

1. Create or update the relevant asset under the correct `assets/` subfolder.
2. Add any supporting notes in `docs/` if they are useful for future maintainers.
3. Verify that the asset is original or properly licensed.
4. Open a review with a brief summary of the content and any distribution restrictions.

## Style

- Use clear filenames.
- Prefer source-friendly organization over ad hoc folders.
- Keep documentation concise and factual.

## Questions

If you are unsure whether something can be added, ask for permission before publishing it here.
